# 1C Accounting for Kazakhstan Examples

This folder contains examples for 1C:Accounting for Kazakhstan.

## Important notes

In 1C:Accounting for Kazakhstan, inventory accounting is based on accounting registers and chart of accounts, not on the `ТоварыНаСкладах` accumulation register used in trade configurations.

Relevant accounts:

- 1300 — Inventory
- 1310 — Raw materials
- 1320 — Finished goods
- 1330 — Goods

These examples are designed with Kazakhstan accounting specifics in mind.
